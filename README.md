Weather Pro
Live Demo

https://weather-webapp-5bvp.onrender.com/

Weather Pro is a full stack weather web application built using Django (backend) and HTML, CSS, JavaScript (frontend).

It fetches real-time weather data using the OpenWeather API and displays detailed weather information in a clean UI.

Features

- Search weather by city name
- Real-time temperature and weather conditions
- Humidity, wind speed, pressure, feels like
- Clean and responsive UI

 Tech Stack

Backend: Django  
Frontend: HTML, CSS, JavaScript  
API: OpenWeather API  
Deployment: Render
